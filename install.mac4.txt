
NECESSARY STEPS FOR TURNING YOUR MAC INTO A SCIENTIFIC COMPUTING PLATFORM  Last updated January, 2010

Instructions for updating a Mac from Leopard to Snow Leopard can be found at http://artsci.wustl.edu/~jgill/snowleopard.txt

Prerequisites, install these first

[ ] Apple Developer Connection Xcode (necessary collection of software tools.  Also available on your installation DVD.)
    http://developer.apple.com/tools/download/ 

[ ] X11 for Mac Os X (also available on your installation DVD)
    http://www.apple.com/downloads/macosx/apple/macosx_updates/x11update2006113.html

[ ] Fink Unix utilities
    http://www.finkproject.org/
    Fink Commander 
    http://finkcommander.sourceforge.net/                                  

Then proceed with everything else as needed.

[ ] MacTeX
    http://tug.org/mactex/
    (Very large, but comprehensive.)

[ ] TeXshop for LaTeX 
    http://www.uoregon.edu/~koch/texshop/obtaining.html

[ ] Other Editors:
    VI: http://sourceforge.net/project/showfiles.php?group_id=213254
        see especially http://tammersaleh.com/posts/the-modern-vim-config-with-pathogen
    Aquamacs: http://aquamacs.org/
    GNU Emacs: Less Mac-specific modificiations than Aquamacs, but easier to customize across platforms 
 	http://emacsformacosx.com/

[ ] R
    http://cran.wustl.edu/sources.html
    (for binaries http://cran.wustl.edu/)
    (or for development versions and tools to allow building of packages http://r.research.att.com/)

[ ] R standalone math libraries
    "The library is not built automatically when R is installed, but can be built 
    in the directory src/nmath/standalone in the R sources: see the file README there."
    in /Users/jgill/Documents/R-2.4.1/src/nmath/standalone, type "make install"
